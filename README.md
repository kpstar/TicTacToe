# Tic Tac Toe
Tic-tac-toe is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game. 
To add this project to android studio.

#### Features
- Single Player with Difficulty levels 
- Two Player

#### Getting started
- In android studio goto file 
- Select new > Import from version control 
- Choose git / github (git option recommanded ) 
- Paste url to repository i.e https://github.com/karansthr/Tic-Tac-Toe.git
- Click on clone.

Play Store https://play.google.com/store/apps/details?id=jetray.tictactoe
